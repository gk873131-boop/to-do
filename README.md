# 📝 Smart To-Do List App

A modern and minimal **To-Do List Web App** built with **Vanilla JavaScript** to manage daily tasks efficiently.

---


## ⚡ Key Highlights

✨ Clean & modern UI
📌 Priority-based tasks (Low / Medium / High)
📅 Due date tracking
⚠️ Overdue task highlighting
🔍 Instant search functionality
🖱️ Drag & drop task reordering
✔️ Mark tasks as completed
💾 Data persistence using Local Storage
📱 Fully responsive design

---

## 🧠 How It Works

* Tasks are saved in **browser localStorage**
* Each task contains:

  * Title
  * Due date
  * Priority
  * Completion status
* Automatically reloads tasks on refresh

---

## 🎯 Features Breakdown

### ➕ Add Task

* Enter task name, select date & priority

### ✏️ Edit Task

* Click **Edit** button to modify task

### 🗑️ Delete Task

* Remove individual tasks anytime

### 🔎 Search Task

* Filter tasks instantly while typing

### 🔄 Drag & Drop

* Reorder tasks easily using mouse

### ✔️ Mark Complete

* Click checkbox or text to toggle completion
